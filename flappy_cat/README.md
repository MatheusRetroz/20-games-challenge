## Project summary

I'm slowly getting back into the challenge... This time, I didn't add extra complexity to the project like I did with Pong.
Overall, the development process was pretty straightforward. The core of the game was built in a single day, and I worked on the rest over the following days whenever I had some free time after work.
 I’m satisfied with the result! (✿◡‿◡)
