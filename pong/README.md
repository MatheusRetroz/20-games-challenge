A clone/remake of the classic arcade game Pong.

## Project summary

The entire process of creating this game was “turbulent” because, unfortunately, due to professional and personal issues, I wasn’t able to devote much time to it—on average, less than 30 minutes a day...and I also ended up creating a very complex structure for the game levels; I know I could have simplified the scripts and scenes, but even so, I’m satisfied with the result!
